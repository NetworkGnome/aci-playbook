# Pre-Deployment Checklist

## ACI Fabric
- [ ] APIC cluster reachable and healthy
- [ ] Firmware version verified
- [ ] Out-of-band management IPs assigned
- [ ] TEP pool defined and non-overlapping

## ISE
- [ ] ISE nodes reachable
- [ ] ERS API enabled
- [ ] pxGrid enabled and certificate trust configured
- [ ] RADIUS shared secret documented

## Integration
- [ ] APIC-ISE pxGrid connection approved
- [ ] EPG-to-SGT mapping planned
- [ ] Test endpoints available for validation

## Network
- [ ] OOB management VLANs provisioned
- [ ] DNS entries created for APIC and ISE nodes
- [ ] NTP synchronized across all devices
